# SQL Advent Calendar 2024

## Overview
This repository contains my solutions to the 24-day SQL Advent Calendar challenge hosted by Interview Master. Each file corresponds to one day of practice and captures query-writing patterns for analytics-focused SQL tasks.

## Repository structure
- `day-01.sql` to `day-24.sql`: one SQL solution per challenge day.
- `README.md`: project documentation.

## Skills practiced
- Filtering and aggregation
- Joins and multi-table analysis
- Window functions
- Subqueries and CTE patterns
- Business-oriented metric calculations

## How to use
1. Open any `day-XX.sql` file.
2. Run the query in your preferred SQL environment.
3. Compare approaches across days to review progress and style.

## Who this is for
- Beginners building daily SQL consistency.
- Analysts preparing for SQL interviews.
- Anyone looking for compact, hands-on SQL exercises.
